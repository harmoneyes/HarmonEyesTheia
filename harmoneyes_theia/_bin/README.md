# Binary Files Directory

This directory contains the platform-specific compiled binaries for the HarmonEyes Theia SDK.

## Expected Files

- `harmoneyes_theia-linux-x86_64.so` - Linux x86_64 binary
- `harmoneyes_theia-macos-arm64.so` - macOS Apple Silicon binary
- `harmoneyes_theia-windows-x86_64.pyd` - Windows x86_64 binary
